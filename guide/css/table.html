# 数据表格

# 表格

表格可以非常快速的部署数据，灵活控制表格样式是必要的。表格不能设置外边距

# 定制表格

除了使用 table 标签绘制表格外，也可以使用样式绘制

样式规则说明
table对应 table
table-caption对应 caption
table-row对表 tr
table-row-group对应 tbody
table-header-group对应 thead
table-footer-group对应 tfoot

# 表格标题

通过 caption-side 可以设置标题位置，值可以设置为 top | bootom

# 内容对齐

水平对齐使用 text-align 文本对齐规则

# 边框间距

/*设置单元格间距，设置间距上下 10px ，左右 50px*/
table {
    border-spacing: 50px 10px;
}

# 边框合并

table {
    border-collapse: collapse;
}

# 隐藏单元格

table {
    empty-cells: hide;
}

# 追加内容

# 基本使用

使用伪类 ::before 向前添加内容，使用 ::after 向后面添加内容

a::after {
    content: " (坚持努力) ";
}

# 提取属性


<style>
  /*使用属性值添加内容，可以使用标准属性与自定义属性*/
  a::after {
    content: attr(href);
  }
</style>
